# 微博去广告与推荐

一个用于去除微博客户端广告和推荐内容的插件。

## 功能

### 广告移除
- 移除开屏广告
- 移除信息流广告
- 移除品牌广告
- 移除图片查看器中的广告
- 移除视频广告
- 移除赞助内容

### 推荐移除
- 移除 Feed 流推荐
- 移除视频推荐
- 移除文章推荐
- 移除用户推荐
- 移除热门推荐
- 移除相似用户推荐
- 移除关注推荐
- 移除故事推荐

## 安装要求
- 已越狱的 iOS 设备
- 安装了 Theos
- 微博客户端

## 安装方法
1. 下载源码
2. 进入项目目录
3. 执行 `make package` 编译
4. 安装生成的 deb 包

## 支持版本
- iOS: 11.0 及以上
- 微博: 最新版本

## 注意事项
- 使用本插件可能会影响部分社交推荐功能
- 建议在使用前备份重要数据
- 如遇问题可以卸载插件恢复原状
